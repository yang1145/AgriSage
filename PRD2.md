## 一、用户系统

| 功能 | 说明 |
|------|------|
| 用户注册 | 本地创建账户，姓名、电话、密码 |
| 用户登录/登出 | 本地验证，无需联网 |
| 角色选择 | 户主 / 家庭成员 / 合作社管理员 / 农技员 |
| 用户切换 | 多用户设备上快速切换身份 |
| 个人信息维护 | 修改姓名、电话、密码 |

---

## 二、地块管理

| 功能 | 说明 |
|------|------|
| 地块列表 | 查看当前用户所有地块卡片，显示名称、面积、状态 |
| 新建地块 | 输入地块名称，手绘边界（地图点击打点），自动计算面积 |
| 地块详情查看 | 显示完整地块信息：边界、面积、土壤、海拔、坡度等 |
| 地块信息编辑 | 修改名称、土壤类型、pH、有机质等 |
| 地块删除 | 户主可删除，删除级联清除关联数据 |
| 地块状态标记 | 闲置 / 种植中 / 已收割 |
| 地块搜索/筛选 | 按名称、状态、面积范围筛选 |

---

## 三、种植周期管理

| 功能 | 说明 |
|------|------|
| 周期创建 | 为地块开启新种植季，选择植期类型（新植/宿根1/2/3年） |
| 宿根自动继承 | 选择宿根时自动带入上季品种、地块信息 |
| 品种选择 | 从预置品种库选择，或手动添加自定义品种 |
| 种植信息录入 | 种植日期、种苗来源、下种量、行距、盖膜情况 |
| 周期列表 | 查看某地块的历史种植季时间轴 |
| 周期详情 | 查看单季完整档案 |
| 周期状态变更 |  growing → harvested → closed |
| 周期删除 | 户主权限 |

---

## 四、农事记录管理

| 功能 | 说明 |
|------|------|
| **施肥记录** | 增删改查：日期、肥料类型、名称、N-P-K、用量、施用方式 |
| **灌溉记录** | 增删改查：日期、方式、水量、水源 |
| **病虫害记录** | 增删改查：发现日期、类型、部位、程度、防治日期、农药、用量、效果 |
| **采收记录** | 增删改查：计划/实际日期、面积、产量、锤度、糖分、价格、糖厂、运费 |
| 农事时间轴 | 按时间顺序展示单周期所有农事记录 |
| 农事筛选 | 按类型（施肥/灌溉/病虫/采收）筛选 |
| 批量关联 | 一次施肥记录可关联多个地块 |

---

## 五、基础数据字典（预置 + 可扩展）

| 功能 | 说明 |
|------|------|
| 品种库浏览 | 查看预置品种信息卡片 |
| 品种搜索 | 按名称、选育单位、适宜区域搜索 |
| 自定义品种添加 | 农户添加本地品种，仅自己可见 |
| 糖厂通讯录浏览 | 按地市查看糖厂信息 |
| 糖厂搜索 | 按名称、区域搜索 |
| 气象站数据查询 | 离线查看预置气象站历史逐日数据 |
| 土壤模板应用 | 选择乡镇自动填充土壤默认值 |

---

## 六、地块影像管理

| 功能 | 说明 |
|------|------|
| 照片上传 | 选择本地照片，关联到指定地块和生育期 |
| 照片时间轴 | 按拍摄日期滑动浏览，生育期分类标签 |
| 照片详情 | 查看原图、拍摄时间、GPS信息 |
| 照片标注 | 在照片上标记异常点（黄叶、倒伏、病斑），记录坐标和标签 |
| 照片删除 | 仅删除数据库记录和标注，不删除系统相册原图 |
| 照片筛选 | 按生育期、拍摄日期范围筛选 |

---

## 七、数据导入导出

| 功能 | 说明 |
|------|------|
| Excel模板下载 | 下载标准导入模板 |
| Excel导入 | 批量导入历史地块/种植/农事数据，错误行高亮 |
| 单地块PDF导出 | 生成含信息表+农事历+照片缩略图的档案 |
| 多地块Excel导出 | 汇总表：农户/面积/品种/产量/糖分 |
| 数据库备份导出 | 导出完整SQLite `.db`文件 |
| 数据库恢复导入 | 从`.db`文件恢复全部数据 |

---

## 八、系统设置

| 功能 | 说明 |
|------|------|
| 数据备份路径设置 | 选择备份文件保存目录 |
| 数据库文件管理 | 查看当前数据库大小、手动备份、清理旧备份 |
| 预置数据更新 | 有网时检测品种库/糖厂库更新（可选） |
| 关于/版本信息 | 显示当前版本、数据量统计 |

## 功能优先级（开发顺序）

```
第一阶段（核心闭环）
├── 用户系统
├── 地块管理（含面积计算）
├── 种植周期管理（含宿根继承）
├── 四大农事记录（施肥/灌溉/病虫/采收）
└── 基础数据字典（预置数据初始化）

第二阶段（体验增强）
├── 地块影像管理
├── 数据导入导出（Excel/PDF/备份）
└── 系统设置

第三阶段（ polish ）
├── 搜索筛选优化
├── 批量操作
└── 界面动效与科幻风格细化
```
