# 桂收 · 甘蔗专用版 Phase 1 功能规格文档

## 1. 产品定位

| 项目 | 说明 |
|------|------|
| 产品名 | 桂收 · 甘蔗专用版 / AgriSage-Cane |
| 阶段 | Phase 1：数据管理（Data Foundation） |
| 目标用户 | 广西甘蔗种植户、合作社管理员、基层农技员 |
| 核心承诺 | 零门槛建档、完全离线可用、数据不出村 |
| 技术栈 | Flask（后端）+ Web 前端（科幻 HUD 风格）+ SQLite（本地单文件） |

---

## 2. 视觉风格定义：甘蔗田里的科幻 HUD

整体视觉语言参考科幻作品中的战术指挥界面（如《流浪地球》控制台、《死亡搁浅》地形扫描 UI），但内容完全贴合农业场景。

| 元素 | 风格要求 |
|------|---------|
| **主色调** | 深空黑（#0a0f14）为底，荧光青（#00f0ff）为主信息色，警报橙（#ff6b35）为异常/预警色，甘蔗绿（#39ff14）为生长/正向状态色 |
| **字体** | 中文：等宽或窄体无衬线（如思源黑体 Condensed）；数字：等宽科技字体（如 DIN、Roboto Mono），带轻微发光效果 |
| **布局** | 模块化卡片 + 网格线辅助 + 边角六边形/斜切装饰，模拟战术面板 |
| **交互反馈** | 按钮点击带"扫描线"划过效果；数据加载时显示"正在同步本地节点"而非传统 loading |
| **地图/地块** | 地块边界以霓虹线框勾勒，不同状态地块以不同脉冲频率闪烁（生长中：呼吸灯；待收割：高亮） |
| **照片浏览** | 时间轴以"轨道"形式呈现，照片以全息投影卡片悬浮展示 |

## 4. 功能模块规格

### 4.1 用户与权限系统

| 功能点 | 规格说明 |
|--------|---------|
| **登录/切换** | 本地用户选择器，无需联网验证；支持密码或免密（本地物理隔离场景） |
| **角色体系** | 户主（全权限）、家庭成员（增查改，无删）、合作社管理员（查看下属汇总）、农技员（查看指定农户并推送文字建议） |
| **用户档案** | 姓名、角色、联系电话；农户可绑定所属合作社 |

---

### 4.2 地块管理（核心模块）

| 功能点 | 规格说明 |
|--------|---------|
| **地块总览面板** | 科幻风格战术地图，以霓虹线框显示所有地块边界；左侧为数据面板，显示总面积、地块数、各状态占比 |
| **新建地块** | 支持两种边界绘制模式：① 地图点击打点自动连线（推荐）② GPS 自动采集（手持设备绕田行走）；系统自动计算面积（亩） |
| **地块信息卡** | 每块地包含：自定义名称、面积、海拔、坡度、坡向、土壤类型（下拉选择，选乡镇后自动填充默认值）、土壤 pH、有机质、土层厚度 |
| **地块状态灯** | 闲置（灰色脉冲）、种植中（绿色呼吸灯）、已收割（橙色常亮） |
| **地块影像轨** | 每个地块独立时间轴，按生育期（萌芽/分蘖/伸长期/成熟期/收获）分类上传照片；支持缩略图轨道滑动浏览 |
| **照片标注** | 在照片上标记异常点（倒伏、病斑、缺素），标注以全息浮标形式呈现 |

---

### 4.3 种植周期管理

| 功能点 | 规格说明 |
|--------|---------|
| **周期创建** | 为地块开启新种植季；选择植期类型（新植蔗 / 宿根 1 年 / 宿根 2 年 / 宿根 3 年+）；宿根类型自动继承上季品种与地块信息，仅需修改日期 |
| **品种选择器** | 下拉搜索广西主推甘蔗品种（桂糖系列、柳城系列、新台糖等），显示品种卡片（产量潜力、糖分、抗逆性、适宜区域）；支持农户手动添加本地品种 |
| **种植档案** | 记录种植日期、种苗来源（自留/糖厂供种/市场购买）、下种量、行距、盖膜情况 |
| **周期时间轴** | 以垂直时间轴展示从种到收的全周期，农事记录自动挂载到对应节点 |

---

### 4.4 农事记录系统

统一入口记录四大类农事，所有记录挂载到具体种植周期。

| 记录类型 | 记录字段 |
|---------|---------|
| **施肥记录** | 日期、肥料类型（尿素/复合肥/磷肥/钾肥/有机肥/叶面肥/其他）、具体名称、N-P-K 含量、施用量（公斤/亩）、施用方式（沟施/撒施/穴施/冲施/叶面喷施）；支持"一次施肥、多地关联"批量录入 |
| **灌溉记录** | 日期、灌溉方式（漫灌/沟灌/滴灌/喷灌/雨养）、水量（立方米/亩）、水源类型（河水/井水/水库/雨水/其他）；雨养模式免填水量 |
| **病虫害记录** | 发现日期、病虫害类型（二点螟/条螟/绵蚜/蓟马/黑穗病/梢腐病/锈病/其他）、危害部位、危害程度（轻/中/重）、防治日期、农药名称、用量、防治效果（待观察/有效/无效）；每种病虫预置离线识别图鉴 |
| **采收记录** | 计划/实际砍收日期、砍收面积、产量（吨）、田间锤度（%）、糖厂化验蔗糖分（后续回填）、收购单价、交售糖厂、运输费用；自动计算亩产值与总收入 |

---

### 4.5 基础数据字典（预置库）

首次安装即内置，无需联网下载。

| 字典 | 内容 |
|------|------|
| **品种库** | 广西近 10 年主推 15-20 个甘蔗品种档案：选育单位、熟性、平均亩产、平均糖分、抗逆评级、建议宿根年限、适宜区域 |
| **土壤模板库** | 按广西 50 个甘蔗重点乡镇预置典型土壤类型（红壤/赤红壤/砖红壤/石灰土/紫色土），选择乡镇后自动填充默认值，用户可修正 |
| **糖厂通讯录** | 广西主要糖厂及下属蔗区站：名称、所属地市/县、服务范围（收购/农技/农机/农资）、联系电话；支持一键拨号 |
| **气象站缓存** | 崇左、来宾、南宁、柳州等主产区气象站近 3 年逐日历史数据（最高温、最低温、降水、日照、湿度），离线可查 |

---

### 4.6 数据导入导出

| 功能 | 规格 |
|------|------|
| **Excel 导入** | 合作社管理员使用标准模板批量导入历史种植数据；系统校验必填字段与格式，错误行高亮提示 |
| **PDF 档案导出** | 单地块完整档案：基本信息表、农事时间轴、照片缩略图矩阵；A4 打印适配，带「桂收」数字水印 |
| **Excel 汇总导出** | 多地块横向汇总：农户、面积、品种、产量、糖分、收入；适配 WPS/Office |
| **数据库备份** | 一键导出 SQLite 单文件（`.db`）到指定目录（U 盘/本地磁盘/网盘同步文件夹），支持完整恢复 |

## 6. 非功能需求

| 类别 | 需求 |
|------|------|
| **离线优先** | 100% 核心功能在无网络环境下可用；有网络时仅用于可选的数据备份同步 |
| **本地化部署** | 双击启动 Flask 服务，浏览器访问 `localhost` 即可使用；无需云服务器、无需域名备案 |
| **单文件数据库** | 所有结构化数据存储于单一 SQLite 文件，便于备份、迁移、恢复 |
| **存储友好** | 文字数据控制在 50MB 以内；照片存储使用系统相册路径引用，不重复保存原图 |
| **性能** | 单表 10 万条记录查询响应 < 200ms；50 张照片时间轴滑动无卡顿 |
| **安全** | 数据库文件支持加密；多用户数据严格隔离 |
| **跨平台** | 支持 Windows 10+、macOS 12+、主流 Linux 发行版 |

## 7. 验收标准（Phase 1）

| 编号 | 验收项 | 通过标准 |
|------|--------|---------|
| AC-01 | 地块建档 | 5 分钟内完成一个地块边界绘制与信息录入，面积计算误差 < 2% |
| AC-02 | 宿根继承 | 创建宿根周期时，自动带入上季品种与地块信息，用户仅需修改日期，操作 ≤ 3 步 |
| AC-03 | 离线可用 | 断开网络后，可完整完成地块查询、农事录入、照片上传、数据导出，重启后数据不丢失 |
| AC-04 | 批量导出 | 导出 50 个地块的 Excel 汇总表，WPS 打开格式正确，耗时 < 10 秒 |
| AC-05 | 预置数据 | 首次安装后，品种库 ≥ 15 条，糖厂通讯录 ≥ 20 条，气象站 ≥ 5 个主产区 |
| AC-06 | 权限隔离 | 用户 A 登录后无法查看、搜索、导出用户 B 的任何数据 |
| AC-07 | 科幻风格 | 所有页面具备统一的 HUD 视觉语言：深色背景、荧光青主色、模块线框、状态呼吸灯 |
| AC-08 | 数据备份 | 导出 `.db` 文件后，在另一台设备导入，数据完整恢复，照片链接可重新关联 |

## 8. 术语表

| 术语 | 说明 |
|------|------|
| 宿根蔗 | 甘蔗收获后，利用地下蔗蔸芽萌发新株，无需重新下种；广西普遍宿根 2-3 年 |
