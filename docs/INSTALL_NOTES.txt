桂收 · 甘蔗专用版 — 安装说明
================================

系统要求
--------
- Windows 10 / Windows 11 (64位)
- 磁盘空间：约 200MB（含运行时依赖）
- 内存：建议 4GB 以上

安装步骤
--------
1. 运行 AgriSage-Setup-1.0.0.exe 安装程序
2. 选择安装路径（默认 C:\Program Files\桂收 · 甘蔗专用版）
3. 勾选是否创建桌面快捷方式
4. 点击「安装」完成部署

首次使用
--------
1. 双击桌面或开始菜单中的「桂收 · 甘蔗专用版」图标
2. 弹出服务管理界面后，点击「启动服务」按钮
3. 服务启动成功后，点击「打开浏览器访问」
4. 使用默认账号登录：
   - 用户名: admin
   - 密码: admin123
5. 首次登录后请及时修改密码

注意事项
--------
- 本软件为离线优先设计，核心功能无需联网即可使用
- 天气数据功能需要网络连接（中国气象局 API）
- 数据库文件存储在安装目录下的 data/ 文件夹中
- 上传的图片存储在 uploads/ 文件夹中
- 备份数据时，只需复制整个 data/ 目录即可

卸载
----
1. 打开「设置」→「应用」，找到「桂收 · 甘蔗专用版」
2. 点击「卸载」即可
3. 卸载不会删除用户数据（data/ 和 uploads/ 目录）
   如需彻底删除，请手动删除安装目录
